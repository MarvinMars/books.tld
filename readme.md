- [fractal](https://fractal.thephpleague.com/)
- [vue-router](https://router.vuejs.org/)
- [vuex](https://vuex.vuejs.org/)
- [sweet alert2 ](https://sweetalert2.github.io/)
- [axios](https://pages.github.com/)
- [vuetify](https://vuetifyjs.com/en/)


## Commands for start 
After clone project
```
composer intall
npm install
php artisan serve 

npm run production

npm run development

php artisan migrate

```
